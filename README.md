<h1>Calculador de media da Faculdade Impacta</h1>

O intuito do projeto é ajudar os alunos da faculdade a ver a sua média no final do bimestre/semestre.

site hospedado em http://mediaimpacta.atwebpages.com/index.php

<h3>Como rodar o codigo na sua maquina</h3>

- Faça o download da pasta no Github
- Instale o xampp para rodar o arquivo PHP no navegador
- Abra o xampp e inicie o apache
- Vá até C:\xampp\htdocs\
- Cole a pasta do Github no local
- Abra o navegador e digite localhost/"nome da sua pasta"
